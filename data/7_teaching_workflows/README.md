# Teaching Workflow Data

This folder contains ready-to-use example datasets for five teaching workflows in the `chem-inf-widgets` Orange add-on.

Generated by:
`data/python_code/build_teaching_workflow_data.py`

Files:
- `01_standardization_smiles.csv`
- `02_diversity_smiles.csv`
- `03_activity_cliffs_vegfr2_ic50.csv`
- `04_qsar_vegfr2_ic50.csv`
- `05_ad_reference_vegfr2_ic50.csv`
- `05_ad_query_vegfr2_ic50.csv`
- `06_mixed_orange_workflows.md`

Suggested workflows:

1. `01_standardization_smiles.csv`
Use with: `Mol Standardizer -> Drug Filter -> Scaffold Analysis`
Goal: show how cleanup and normalization change downstream chemistry.

2. `02_diversity_smiles.csv`
Use with: `Fingerprint Generator -> Diversity Picker -> Scaffold Analysis`
Goal: compare diverse subset selection methods on a medium-sized library.

3. `03_activity_cliffs_vegfr2_ic50.csv`
Use with: `Mol Standardizer -> Activity Cliff Finder -> Molecular Viewer`
Goal: identify highly similar compounds with large potency differences.

4. `04_qsar_vegfr2_ic50.csv`
Use with: `Mol Standardizer -> Mol Descriptors 2 or Fingerprint Generator -> QSAR Regression`
Goal: build and diagnose a regression model on real ChEMBL VEGFR2 data.

5. `05_ad_reference_vegfr2_ic50.csv` and `05_ad_query_vegfr2_ic50.csv`
Use with: `Mol Standardizer -> descriptor widget -> Applicability Domain`
Goal: fit AD on a reference set and probe a separate query set.

Notes:
- The VEGFR2 datasets were retrieved from the official ChEMBL API endpoint for target `CHEMBL279` with `standard_type=IC50`.
- Rows are deduplicated by molecule, preferring the highest available `pChEMBL` and otherwise the lowest `IC50`.
- The AD files are deterministic row splits of the same curated VEGFR2 set.
- `06_mixed_orange_workflows.md` contains five additional classroom examples that combine chemoinformatics widgets with core Orange widgets such as projections, clustering, outlier detection, box plots, and heat maps.
