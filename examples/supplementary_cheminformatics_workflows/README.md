# Supplementary Cheminformatics Workflow Example Data

These small datasets are designed for teaching. They are not curated research datasets and should be used only for classroom demonstrations.

Most files contain `name` and `smiles` columns plus one or more teaching annotations. Use them with File, Mol Standardizer, Fingerprint Generator, Cyclic Registry Fingerprint, Similarity Search, Substructure Search, Diversity Picker, Drug Filter, Scaffold Analysis, Activity Cliff Finder, Matched Molecular Pairs, or visualization widgets.
