# QSAR Studio examples

- `qsar_dataset_builder_demo.csv` demonstrates raw bioactivity curation.
- `qsar_model_hub_demo.csv` is a descriptor/fingerprint-like table ready for `QSAR Model Hub`.
- `qsar_predictions_demo.csv` is ready for `QSAR Validation Dashboard`.

- `qsar_ad_explanation_demo.csv`: small descriptor/fingerprint table for AD Workbench and Model Explanation.
- `qsar_ad_query_demo.csv`: query/external rows with deliberate descriptor-space outliers.
