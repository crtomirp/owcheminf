# Example Data: ChEMBL Bioactivity Curation

This folder contains small teaching datasets for the ChEMBL Bioactivity Curation package.

## Files

- `chembl_target_search_examples.csv` — target search prompts for classroom discussion.
- `chembl_bioactivity_curation_demo.csv` — simplified synthetic bioactivity records for practicing curation decisions.
- `chembl_qsar_curated_example.csv` — a small curated example table suitable for downstream descriptor/fingerprint demonstrations.

## Important note

The demo bioactivity records are simplified for teaching. They are not intended to represent a real ChEMBL target dataset. For real research, retrieve live ChEMBL records and document the query, filters, target identifier, and retrieval date.
