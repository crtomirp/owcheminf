# Medicinal Chemistry Interpretation of Cliff Pairs

## Context

Activity cliffs can reveal important substituent effects, steric effects, hydrogen-bonding changes, or electronic effects.

## Intended learning outcomes

Students can:

- interpret cliff pairs as SAR evidence,
- propose plausible structural explanations,
- distinguish evidence from speculation.

## Orange workflow

```text
File → Activity Cliff Finder → Cliff Pairs → Data Table
```

Optional:

```text
File → Cyclic Registry Fingerprint → Matched Registry Entries
```

## Tasks

1. Choose three cliff pairs.
2. Compare their SMILES or structures.
3. Identify the most obvious substituent difference.
4. Propose a chemical reason for the activity change.
5. Mark your explanation as strong, tentative, or speculative.

## Guiding questions

1. Is the change electronic, steric, polar, lipophilic, or scaffold-related?
2. Could protonation state matter?
3. Could the difference reflect assay noise?
4. What experiment or computation would you use to test the hypothesis?

## Expected output

Students should produce short SAR interpretations for selected cliff pairs.
