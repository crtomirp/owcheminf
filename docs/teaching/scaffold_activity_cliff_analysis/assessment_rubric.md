# Assessment Rubric: Scaffold and Activity Cliff Analysis

| Criterion | 0 points | 1 point | 2 points | 3 points |
|---|---|---|---|---|
| Data preparation | Dataset missing or unusable | Dataset loaded with major issues | Dataset mostly correct | Dataset clean and documented |
| Scaffold analysis | Not performed | Performed but not interpreted | Correct basic interpretation | Strong chemical interpretation |
| Scaffold split reasoning | Not discussed | Random/scaffold split confused | Correct basic explanation | Clear explanation of leakage and generalisation |
| Activity cliff detection | Not performed | Settings inappropriate | Reasonable settings and output | Thoughtful threshold choice and interpretation |
| Medicinal chemistry reasoning | Absent | Superficial | Correct for examples | Insightful SAR discussion |
| QSAR connection | Absent | Weak | Clear basic connection | Strong integration with validation and AD |
| Reproducibility | Missing | Partial | Mostly complete | Complete and transparent |

Suggested total: 21 points.
