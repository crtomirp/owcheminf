# Teaching Chemistry with Orange Data Mining

**Pedagogical guidelines, classroom patterns, and ready-to-use examples for cheminformatics teaching.**

This folder contains a modern teacher-facing booklet in DOCX/PDF form. It complements the student worksheet packages in `docs/teaching/`.

## Files

- `orange_in_chemistry_pedagogy_booklet.docx` - editable designed booklet.
- `orange_in_chemistry_pedagogy_booklet.pdf` - shareable PDF export.

## Core message

Orange should be taught as a way to express chemical reasoning as a transparent computational workflow. Students should learn not only how to connect widgets, but how to justify data curation, preprocessing, feature representation, validation, interpretation, and limitations.

## Covered topics

1. Teaching philosophy for Orange in chemistry
2. TPACK, Bloom, and SAMR alignment
3. Modern lesson architecture
4. Core cheminformatics workflow patterns
5. Map of the included teaching packages
6. Ten classroom-ready examples
7. Assessment rubrics
8. Accessibility and classroom management
9. Troubleshooting and reproducibility
10. Implementation roadmap

## Suggested use

Use the booklet for teacher preparation, curriculum planning, and explaining the pedagogical rationale behind the worksheet packages.
