# QSAR Studio Teaching Add-on

This module introduces the Phase 4 **QSAR Studio** workflow, beginning with the
`QSAR Dataset Builder` widget.

## Learning goals

Students learn to:

1. distinguish raw bioactivity records from QSAR-ready compounds;
2. convert IC50/Ki/Kd/EC50 concentrations to pActivity values;
3. reject ambiguous activity records;
4. aggregate duplicate compound measurements;
5. document dataset curation decisions before modeling.

## Core workflow

```text
Molecule Import Hub → Molecule QC Dashboard → QSAR Dataset Builder → QSAR Regression
```
