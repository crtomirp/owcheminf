# General Assessment Rubric

| Criterion | 0 points | 1 point | 2 points | 3 points |
|---|---|---|---|---|
| Workflow construction | Missing or incorrect | Partly working | Correct basic workflow | Correct and well documented workflow |
| Data handling | No clear input/output | Some outputs checked | Correct tables inspected | Outputs exported and interpreted reproducibly |
| Chemical interpretation | No interpretation | Superficial comments | Correct chemical explanation | Insightful structure-property reasoning |
| Limitations | Not discussed | Generic limitation | Relevant limitation identified | Limitations linked to workflow choices |
| Reporting | Missing | Minimal | Clear | Clear, reproducible, and well structured |

Suggested score: 15 points per worksheet.
