# Supplementary Cheminformatics Workflows Teaching Package

This package adds worksheets for parts of the `chem-inf-widgets` collection that were not fully covered by the earlier teaching packages.

Earlier packages focused on:

- Cyclic Registry Fingerprint
- QSAR widgets
- Applicability Domain
- Molecular Standardization and Data Cleaning
- ChEMBL Bioactivity Curation
- Scaffold and Activity Cliff Analysis

This supplementary package covers:

1. molecular similarity search,
2. substructure and SMARTS search,
3. diversity picking,
4. drug-likeness and filtering,
5. SDF import/export and data provenance,
6. molecule visualization,
7. R-group decomposition,
8. matched molecular pairs,
9. reaction enumeration,
10. pharmacophore fingerprint searching,
11. pair/compound detail inspection,
12. an integrated capstone workflow.

## Recommended audience

Advanced undergraduate or MSc students in chemistry, pharmacy, medicinal chemistry, cheminformatics, computational chemistry, or chemical data science.

## Main pedagogical goal

Students learn how cheminformatics workflows support practical chemical decision-making, not only model building. The worksheets emphasize interpretation, uncertainty, reproducibility, and the relationship between molecular structure and data representations.
