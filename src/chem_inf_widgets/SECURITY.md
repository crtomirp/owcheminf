# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |

## Reporting a Vulnerability

Please report security vulnerabilities by opening a [GitHub issue](https://github.com/crtomirp/chem-inf-widgets/issues) with the label `security`, or by contacting the maintainer directly at crtomir.podlipnik@gmail.com.

You can expect an acknowledgement within 7 days and a fix or mitigation plan within 30 days for confirmed vulnerabilities.
