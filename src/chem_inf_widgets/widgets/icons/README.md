## Icon Groups

- `input_output/`: import/export widgets such as SDF readers and writers
- `editors_viewers/`: structure drawing, inspection, pair viewing, compound cards, 3D viewing
- `standardization_filtering/`: standardization, filtering, substructure search, similarity-style search
- `descriptors/`: fingerprints, Mordred, PaDEL, ISIDA, descriptor generation
- `analysis/`: SAR, scaffold, matched pairs, diversity, applicability domain
- `data_retrieval/`: ChEMBL and related retrieval widgets
- `modeling/`: QSAR and predictive modeling widgets
- `reactions/`: reaction viewing, transformation and enumeration
